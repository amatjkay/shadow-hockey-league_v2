## 🛑 Блокеры и риски

| Риск                                              | Влияние                         | Мера смягчения                                                   |
| ------------------------------------------------- | ------------------------------- | ---------------------------------------------------------------- |
| Перенос тестов может сломать импорты              | Блокирует все последующие этапы | Делать первым (Этап 3.2), тщательно проверить импорты            |
| Рефакторинг формулы изменит результаты расчёта    | Несоответствие текущим данным   | Написать тесты на текущие ожидаемые значения перед рефакторингом |
| Flask-Limiter может потреблять дополнительную RAM | 1GB VPS лимит                   | Протестировать локально, настроить rate limits консервативно     |
| Alembic миграции могут конфликтовать              | Блокирует деплой                | Проверить текущие миграции, создать миграции последовательно     |
| CSRF может сломать API запросы                    | API перестанет работать         | CSRF только для админки, API использует API Keys                 |
| Расхождение тестов (72 vs 48)                     | Непонимание реального покрытия  | Аудит при переносе (Этап 3.2), зафиксировать точное количество   |
