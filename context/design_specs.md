# Design Specifications — Shadow Hockey League v2

**Дата:** 3 апреля 2026 г.
**Статус:** 🟡 Ожидание анализа дизайнера

## 🎨 Текущий UI (из README.md)

- **Единая таблица «Рейтинг лиги»** с колонками: страна, место, очки, участник, кубки, расчёт
- **Анимированный градиент** для топ-10 строк (колонки Место, Очки, Участник)
- **Тандемы:** префикс `Tandem:` скрыт, бейдж «тандем» отображается при наличии запятой в имени
- **Блок «Как считаются очки»** — свёрнут по умолчанию
- **Детальный расчёт** по наградам — раскрывается по нажатию
