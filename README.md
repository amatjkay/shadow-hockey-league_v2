# Shadow Hockey League

Веб-приложение на **Flask**: таблица менеджеров (соло и тандемы) с кубками и **рейтинг по очкам** на одной странице.

## Возможности

- Единая таблица **«Рейтинг лиги»** (колонки: **страна** · **Место** · **Очки** · **Участник** · **Кубки** · **Расчёт**). Детальный расчёт по наградам — в последней колонке, раскрывается по нажатию.
- Правила начисления очков — в блоке **«Как считаются очки»** (свернут по умолчанию).
- Для **топ-10** строк анимированный градиент на колонках **Место**, **Очки** и **Участник** синхронизирован (`background-attachment: fixed` и общий `@keyframes league-top10-sheen`).
- **Тандемы:** префикс `Tandem:` в интерфейсе не показывается (остаётся только состав, например `Vlad, whiplash 92`). Бейдж **«тандем»** выводится, если в **исходном** имени записи есть запятая (пара участников в данных).
- Старый URL `/rating` перенаправляет на главную с якорем `/#rating`.

## Расчёт очков

Логика в модуле `data/rating.py`: за каждую награду берётся **база** (лига и тип кубка) × **множитель сезона**, результат округляется, затем суммируется по строке участника.

- Сезоны: **s23/24** ×1,15 · **s22/23** ×1,00 · **s21/22** ×0,85  
- База: лига 1 / лига 2 — TOP1–3, Best regular, Round 3, Round 1 (см. код и блок правил на сайте). Toxic = 0.

Данные менеджеров и HTML кубков: `data/managers_data.py`.

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

По умолчанию сервер: `http://127.0.0.1:5000/`.

Порт: `python app.py --port 8080`.

## Тесты

```bash
python -m unittest
```

## Структура (основное)

| Путь | Назначение |
|------|------------|
| `app.py` | Приложение Flask, маршруты `/` и редирект `/rating` |
| `data/managers_data.py` | Список менеджеров и достижений |
| `data/rating.py` | Построение таблицы рейтинга для шаблона |
| `templates/index.html` | Главная страница |
| `static/css/style.css` | Стили |

При необходимости БД и сиды: `models.py`, `seed_db.py` (отдельный сценарий наполнения).
