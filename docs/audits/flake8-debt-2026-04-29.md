## 3. Action plan

The 9 hidden codes split into three buckets by safety / value:

### High-value real bugs (do first)
- **`F821` × 2** in `tests/integration/test_cache_invalidation.py:84-86` —
  references to undefined `leagues` / `seasons` from a half-applied
  refactor. The test currently passes only because it never reaches that
  branch. **Fix imports or remove dead code path.**

### Code-quality cleanup (do next)
- **`F841` × 4** — drop the unused locals.
- **`E711` × 2 / E712 × 1** — replace `== None` / `== True` with `is`.
- **`E741` × 3** — rename `l` → something descriptive.

### Cosmetic (do last, ideally one mass commit)
- **`W503` × 9** — disable globally in `.flake8` (PEP 8 itself
  acknowledges that "line break before binary operator" is now the
  preferred style).
- **`F541` × 7** (3 disappear after PR #41) — convert `f"..."` with no
  placeholders back to plain strings.
- **`E402` × 5** — `wsgi.py:15` is justified (sys.path manipulation
  before imports), so add a `# noqa: E402` there. The remaining 4 in
  scripts/ can be reordered.

## 4. Suggested cleanup PRs

1. **Real bugs** (`F821`) — single small PR, high signal.
2. **Code-quality** (`F841`, `E711`, `E712`, `E741`) — single PR, no
   behaviour change.
3. **Cosmetic** (`W503`, `F541`, `E402`) — single PR + drop the
