## 1. Резюме проекта

Shadow Hockey League — веб-приложение для управления фентезийной хоккейной лигой, рейтинга менеджеров и отслеживания достижений. Приложение предоставляет публичный рейтинг и админ-панель для управления данными.

**Production URL:** https://shadow-hockey-league.ru/

## 10. Глоссарий

| Термин           | Определение                           |
| ---------------- | ------------------------------------- |
| **Менеджер**     | Участник лиги (соло или тандем)       |
| **Тандем**       | Команда из двух менеджеров            |
| **Достижение**   | Награда/кубок за сезон                |
| **Лига 1**       | Высший дивизион                       |
| **Лига 2**       | Второй дивизион                       |
| **Сезон**        | Период соревнований (например, 24/25) |
| **TOP1**         | Первое место в лиге                   |
| **Best regular** | Лучший в регулярном чемпионате        |
