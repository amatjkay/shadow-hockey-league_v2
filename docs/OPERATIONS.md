# Operations, Security & Monitoring Guide

## 1. Admin Panel Administration
### First Access
1. **Create Admin User**:
   ```bash
   source venv/bin/activate
   python scripts/create_admin.py
   ```
2. **Login**: `https://shadow-hockey-league.ru/admin/`

### Key Features
- **Tandem Warning**: Alerts when entering comma-separated names.
- **Bulk Operations**: Create achievements for multiple managers at once.
- **Audit Logs**: Track all admin actions.

## 2. Security Configuration
### Authentication
- **Admin**: Flask-Login (session-based).
- **API**: API Key (hashed in database).

### Protection
- **CSRF**: Flask-WTF protection for all forms.
- **Rate Limiting**: 100 req/min for API keys.
- **SSL**: Automated Let's Encrypt certificates.
