# 🛠 Админ-панель Shadow Hockey League

Руководство по использованию админ-панели для управления лигой.

---

## 🎯 Обзор

Админ-панель предназначена для управления данными лиги:
- **Страны** (Country) — флаги и коды стран
- **Менеджеры** (Manager) — участники лиги
- **Достижения** (Achievement) — награды и кубки
- **Пользователи** (Admin Users) — администраторы

**URL:** `https://shadow-hockey-league.ru/admin/`

---

## 🔐 Вход в админ-панель

### Первый вход

1. **Создайте первого админа** (только один раз):

   ```bash
   # На VPS
   cd /home/shleague/shadow-hockey-league_v2
   source venv/bin/activate
   python scripts/create_admin.py
   ```

2. **Введите данные:**
   - Username: `admin` (или любой другой)
   - Password: `ваш-пароль` (минимум 6 символов)

3. **Войдите в панель:**
   - Откройте `https://shadow-hockey-league.ru/admin/`
   - Введите логин и пароль
   - Нажмите "Login"

---

## 📊 Управление данными

### Страны (Countries)

**Что можно делать:**
- ✅ Добавить новую страну
- ✅ Редактировать код страны
- ✅ Изменить путь к флагу
- ✅ Удалить страну (если нет менеджеров)

**Поля:**
- `code` — 3-буквенный код (например, `RUS`, `BEL`)
- `flag_path` — путь к файлу флага (например, `/static/img/flags/rus.png`)

---

### Менеджеры (Managers)

**Что можно делать:**
- ✅ Добавить нового менеджера
- ✅ Редактировать имя
- ✅ Изменить страну
- ✅ Удалить менеджера (достижения удалятся каскадом)

**Поля:**
- `name` — имя менеджера (уникальное)
- `country_id` — страна (выпадающий список)

**Тандемы:**
- Для создания тандема используйте запятую в имени: `Vlad, whiplash 92`
- Префикс `Tandem:` добавляется автоматически (не нужно вводить)

---

### Достижения (Achievements)

**Что можно делать:**
- ✅ Добавить новое достижение
- ✅ Редактировать тип, лигу, сезон
- ✅ Изменить менеджера
- ✅ Удалить достижение

**Поля:**
- `achievement_type` — тип достижения (`TOP1`, `TOP2`, `TOP3`, `BEST`, `R3`, `R1`)
- `league` — лига (`1` или `2`)
- `season` — сезон (например, `24/25`, `23/24`)
- `title` — название (например, `TOP1`, `Best regular`)
- `icon_path` — путь к иконке
- `manager_id` — менеджер (выпадающий список)

**Фильтры:**
- По лиге (`1`, `2`)
- По сезону (`24/25`, `23/24`, etc.)
- По типу (`TOP1`, `TOP2`, etc.)

---

### Пользователи (Admin Users)

**Что можно делать:**
- ✅ Добавить нового админа
- ✅ Изменить логин
- ✅ Сменить пароль
- ✅ Удалить админа (нельзя удалить последнего)

**Поля:**
- `username` — логин (уникальный)
- `password_hash` — пароль (хешируется автоматически)

---

## 🔄 Инвалидация кэша

**Автоматическая инвалидация:**
- При создании/редактировании/удалении менеджера
- При создании/редактировании/удалении достижения
- При создании/редактировании/удалении страны

**Кэш обновляется:**
- Главная страница (`/`) кэшируется на 5 минут
- При изменении данных кэш сбрасывается автоматически

---

## 🛡 Безопасность

### Требования к паролю
- Минимум 6 символов
- Рекомендуется: буквы + цифры + спецсимволы

### Защита админ-панели
- Все маршруты `/admin/*` защищены
- Требуется аутентификация
- Сессии используют `SECRET_KEY`

### Рекомендации
1. **Смените пароль по умолчанию** (если создавали через скрипт)
2. **Используйте HTTPS** (настроено на VPS)
3. **Не делитесь логином/паролем**

## 📝 Примеры использования

2. Перейдите в **Managers** → **Create**

## 🐛 Устранение неполадок

### Не могу войти

**Проблема:** "Invalid username or password"

**Решение:**
1. Проверьте раскладку клавиатуры
2. Убедитесь, что Caps Lock выключен
3. Создайте нового админа:
   ```bash
   python scripts/create_admin.py
   ```

### Ошибка при удалении страны

**Проблема:** "Cannot delete country with X manager(s)"

**Решение:**
1. Сначала удалите или переназначьте менеджеров этой страны
2. Затем удалите страну

### Кэш не обновляется

**Проблема:** Изменения не отображаются на главной

**Решение:**
1. Подождите 5 минут (истечение TTL кэша)
2. Или перезапустите приложение:
   ```bash
   systemctl restart shadow-hockey-league
   ```
