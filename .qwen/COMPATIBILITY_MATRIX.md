## 🔄 Миграция между LLM

### Из Qwen в GPT-4/Gemini/Claude

1. **Скопируй** `.qwen/` директорию
2. **Скопируй** `.windsurfrules`
3. **Измени** `.qwen/config.json`:
   ```json
   {
     "llm_provider": { "current": "openai_gpt4" },
     "operation_mode": { "current": "direct_only" }
   }
   ```
4. **Готово!** Универсальное ядро работает

### Из GPT-4/Gemini/Claude в Qwen

1. **Скопируй** `.qwen/` директорию
2. **Скопируй** `.windsurfrules`
3. **Измени** `.qwen/config.json`:
   ```json
   {
     "llm_provider": { "current": "qwen_code_companion" },
     "operation_mode": { "current": "full_subagents" }
   }
   ```
4. **Готово!** Субагенты активны
