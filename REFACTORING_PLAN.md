## 🎯 Общая информация о проекте

**Shadow Hockey League** — веб-приложение на Flask для отслеживания рейтинга менеджеров хоккейной лиги на основе их достижений (кубков).

### Архитектура системы

```
┌─────────────────────────────────────────────────────────────┐
│                        Browser                              │
│                    (index.html + CSS)                       │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼ HTTP/HTTPS
┌─────────────────────────────────────────────────────────────┐
│                    Flask Application                        │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐ │
│  │  app.py     │  │  config.py  │  │  services/          │ │
│  │  (routes)   │  │  (settings) │  │  - rating_service.py│ │
│  │             │  │             │  │  - api.py (REST)    │ │
│  │             │  │             │  │  - validation_svc.py│ │
│  └─────────────┘  └─────────────┘  └─────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼ SQLAlchemy
┌─────────────────────────────────────────────────────────────┐
│                    SQLite Database                          │
│  ┌──────────┐  ┌──────────┐  ┌──────────────────┐          │
│  │ countries│  │ managers │  │ achievements     │          │
│  │ (code,   │◄─│ (name,   │◄─│ (type, league,   │          │
│  │  flag)   │  │  country)│  │  season, title)  │          │
│  └──────────┘  └──────────┘  └──────────────────┘          │
└─────────────────────────────────────────────────────────────┘
```

### Ключевые компоненты

| Компонент                        | Назначение                                        | Зависимости                    |
| -------------------------------- | ------------------------------------------------- | ------------------------------ |
| `app.py`                         | Flask приложение, маршруты, security headers      | `models`, `services`           |
| `models.py`                      | SQLAlchemy модели (Country, Manager, Achievement) | `db`                           |
| `config.py`                      | Конфигурация через .env                           | `python-dotenv`                |
| `services/rating_service.py`     | Расчёт очков: база × сезон                        | `models`                       |
| `services/api.py`                | REST API CRUD                                     | `models`, `validation_service` |
| `services/validation_service.py` | Валидация данных                                  | `models`                       |
| `seed_db.py`                     | Наполнение БД                                     | `data/managers_data`           |
| `templates/index.html`           | Главная страница с рейтингом                      | Jinja2                         |
| `static/css/*.css`               | Модульные стили (11 файлов)                       | —                              |
