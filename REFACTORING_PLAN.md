## 🎯 Общая информация о проекте

**Shadow Hockey League** — веб-приложение на Flask для отслеживания рейтинга менеджеров хоккейной лиги на основе их достижений (кубков).

### Текущая архитектура

```
shadow-hockey-league_v2/
├── app.py                      # Flask приложение (Application Factory)
├── models.py                   # SQLAlchemy модели (Country, Manager, Achievement)
├── config.py                   # Конфигурация с поддержкой .env
├── services/
│   ├── rating_service.py       # Расчёт рейтинга
│   ├── api.py                  # REST API Blueprint
│   └── validation_service.py   # Валидация данных
├── scripts/
│   └── validate_db.py          # Скрипт валидации БД
├── data/
│   └── managers_data.py        # Исходные данные
├── templates/
│   ├── index.html              # Главная страница
│   └── error.html              # Страница ошибки
├── static/css/                 # Модульные CSS файлы (11 файлов)
├── migrations/                 # Alembic миграции
└── tests.py                    # Unit и интеграционные тесты
```

#### 1.1 Исправление дублирования Max Trufanov
**Файл:** `data/managers_data.py`

**Проблема:**
```python
# Строка 16: Первый Max Trufanov (2 достижения)

    '<img src="/static/img/cups/clap.svg" title="Shadow 1 league Round 1 s22/23">',
    '<img src="/static/img/cups/clap-b.svg" title="Shadow 1 league Round 3 s21/22">'
]),

# Строка 44: Второй Max Trufanov (3 достижения)
Manager("Max Trufanov", "/static/img/flags/ua.png", [
    '<img src="/static/img/cups/clap.svg" title="Shadow 1 league Round 1 s22/23">',
    '<img src="/static/img/cups/clap-b.svg" title="Shadow 1 league Round 3 s21/22">',
    '<img src="/static/img/cups/clap.svg" title="Shadow 2 league Round 1 s24/25">'
])
```

**Решение:**
- [ ] Объединить записи в одну с 3 достижениями

**Подзадачи:**
